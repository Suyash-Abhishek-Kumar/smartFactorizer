Smart Factorizer

The program prime factorizes the user input number and then compresses each prime factor separately.
It also breaks down any prime numbers it encounters while factorizing the original number.
Finally, it give the user 3 variants of the answer: +, -, =
+ means the factorizing added 1 to any new prime factors encountered
- means the factorizing subtracted 1 from any new prime factors encountered
= means the factorizing added or subtracted 1 to/from any new prime factors encountered based on the coded logic
